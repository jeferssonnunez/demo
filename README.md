## Getting Started
### Gradle Commands

**Run in Local JVM**
```
./gradlew bootRun
```

**[Remote debug](https://www.jetbrains.com/help/idea/tutorial-remote-debug.html) in Local JVM**
```
./gradlew bootRun --debug-jvm
```

**Compile, Run Unit Tests, and Lint**
```
./gradlew build
```
